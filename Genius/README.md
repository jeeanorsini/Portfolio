# Jogo da Memória Estilo Genius - Versão Aprimorada

Este é um projeto de jogo da memória inspirado no clássico **Genius**, desenvolvido inicialmente como parte do curso "Criando seu Jogo de Memória Estilo Genius" ministrado por **Gabriela Pinheiro**, uma das tutoras da **DIO**.

## Sobre o Projeto

O projeto original consiste em um jogo de memória com padrões de cores que os usuários devem reproduzir na ordem correta. A versão base fornecida no curso serviu como ponto de partida para estudo e prática de **JavaScript, HTML e CSS**, além de conceitos de lógica, manipulação de DOM e programação orientada a eventos.

## Melhorias Implementadas

Durante o desenvolvimento, foram implementadas melhorias para aprimorar a experiência do usuário e a estética do jogo:

- **Visual mais atraente**: adição de sombras, bordas arredondadas e cores mais harmonizadas.
- **Feedback visual aprimorado**: alterações no hover e animações para indicar cliques e sequência correta/incorreta.
- **Código mais organizado**: refatoração de funções para maior clareza e manutenção futura.
- **Compatibilidade e responsividade**: ajustes para que o jogo funcione bem em diferentes tamanhos de tela.

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**

## Reproduzindo o projeto

1. Caso queira facilidade em reproduzir clique aqui => [Genius](https://jeeanorsini.github.io/Portfolio/Genius/)
2. Caso queira o código para efetuar melhorias fique a vontade em clonar meu código. => [Clonando Repositório](https://docs.github.com/pt/repositories/creating-and-managing-repositories/cloning-a-repository).

## Como Jogar

1. Clique no botão “OK” para iniciar o jogo.  
2. Observe a sequência de cores apresentada.  
3. Reproduza a sequência clicando nas cores na ordem correta.  
4. O jogo adiciona um nível a cada sequência correta.  
5. Tente manter a sequência pelo maior número de rodadas possível!

## Conclusão

Este projeto foi uma oportunidade de aprender conceitos essenciais de programação web e também de praticar **melhorias criativas** em um projeto existente. O objetivo foi não apenas replicar o jogo original, mas **incrementar funcionalidades e design**, mostrando evolução prática do aprendizado.
